# ADHD Vibe Coding App — Competitor Research

## Core Concept
Mobile web app for ADHD devs who vibe code. Voice-first capture + AI organization + document TTS reading.

---

## ADHD Voice/AI Note Apps

### F0cusly ($24.99 one-time)
- On-device AI (BitNet), Apple Speech, Apple Vision OCR
- Voice + photo + text capture
- AI search with citations
- Auto task extraction + CBT micro-steps
- Pomodoro timer
- 2-tap capture, screen stays awake
- Privacy-first, offline capable

### Recordo
- Voice/text/photo AI organizer
- AI personalities (gentle → direct)
- Momentum Mode for task paralysis
- MCP server → ChatGPT/Claude integration
- Brain dump → structured tasks
- Built specifically for ADHD

### Calma
- Voice brain dump → prioritized tasks
- AI mood journaling
- Pixel habit tracking
- Clean, low-stimulation dark UI
- 2-minute brain dumps

### Quillo
- Voice → tasks, events, shopping lists, reminders
- 99% accuracy, context-aware AI
- One 60-sec ramble → 5+ organized actions
- Apple Watch support
- Routes to Reminders, Todoist, Notion, etc.

### WhisperPlan
- Voice-first task manager
- OpenAI Whisper + Gemini AI
- Project auto-detection from context
- Energy level matching
- 2-second quick capture
- Forgiving rescheduling

### Keysha AI
- Free iOS app
- Tasks, calendar, email — one place
- Voice-first, adapts to energy shifts
- Natural language capture
- Calendar integrations (Google, Outlook, Apple)

### Gaveki
- Brain dump → AI picks 3 tasks
- Voice coach (5 personalities: Sergeant Focus, Dr. Calm, etc.)
- Body doubling rooms
- AI task breakdown
- Distraction shield
- Peak energy scheduling
- Weekly progress reports read aloud

### ConvoAct
- Claude-powered
- AI task breakdown with energy costs
- Mood + medication + hydration tracking
- Burnout detection (10 factors)
- Siri voice commands
- 48 AI coach configurations

### tickk (Free, PWA, offline)
- Voice-first, AI-free NLP (compromise.js)
- Auto-categorizes into tasks/notes/reminders
- Privacy-first (local storage, no AI servers)
- Focus Mode, Command Palette (⌘K)

### Ottex (Developer-focused)
- System-wide macOS dictation
- 150+ WPM speaking vs 40 WPM typing
- Custom dictionary for technical terms
- Voice corrections ("scratch that", "actually")
- IDE support (VS Code, Cursor, JetBrains)
- App-specific modes (coding vs casual)

### Seraph ($10/mo)
- Voice dictation for IDEs
- Prompt shortcuts panel
- AI assistant integration
- File injection templates

### Focused (befocused.ai)
- Real-time distraction monitoring
- Voice coaching nudges
- Focus analytics
- Deep Work Mode (AI monitors screen)

### Hugo (macOS)
- AI evaluates every tab/app in real-time
- No blocklist needed
- Project memory (remembers which apps/tabs belong)
- Session modes: Deep Work, Pomodoro, Research, Break

### VoiceType (voicetype.com)
- 360 WPM dictation across apps
- Auto-format and tone for context
- Whisper Mode for discreet dictation
- 35+ languages, 99.7% accuracy
- Works in Slack, Notion, Gmail, etc.

### Whisper Memos
- Apple Watch + lock screen widgets
- OpenAI Whisper or ElevenLabs Scribe
- AI summaries + custom prompts
- Routes to Notion, Todoist, Things 3, etc.

## TTS/Reader Apps (ElevenReader inspiration)

### ElevenReader
- Upload PDFs, docs, web links, paste text
- 32+ languages, iconic voice library
- Speed control: 0.25x to 3x
- Text highlighting (dyslexia/ADHD support)
- Personal pronunciation dictionary
- Bookmarks, notes, clip & share
- Sleep timers
- Sync across devices

### Other TTS readers in market
- Various epub/PDF readers with TTS
- Most lack: custom voice design, pronunciation dictionary, synced highlighting

## Key Patterns from Competitors

### ADHD-Specific UX Patterns
1. **3-item daily cap** (Gaveki: pick 3 tasks, Calma: priority focus)
2. **Brain dump first, organize later** (tickk, Recordo, Calma)
3. **Gamification without streak-punishment** (gentle visuals, no guilt)
4. **AI task breakdown** (ConvoAct, Gaveki, WhisperPlan)
5. **Energy-aware scheduling** (ConvoAct, Gaveki)
6. **Voice coach personas** (Gaveki: 5 personalities)
7. **Calm, low-stimulation UI** (dark mode, minimal clutter)

### Voice Capture Patterns
1. **2-tap capture** (F0cusly)
2. **Ramble for up to 5 minutes** (Quillo, WhisperPlan)
3. **Auto-categorization** (tickk AI-free, Recordo AI)
4. **Apple Watch + lock screen widgets** (Whisper Memos)

### TTS Reader Patterns (ElevenReader)
1. **Custom voice design** (key differentiator)
2. **Pronunciation dictionary** (user corrections)
3. **Synced highlighting** (follow text as it reads)
4. **Speed control: 0.25x to 4x** (fine-grained)
5. **Sleep timer + bookmarks + notes**

## Market Gap Identified

**No app combines:**
- ADHD brain-dump/voice-first capture (F0cusly, Recordo, Calma)
- WITH
- AI TTS document reading (ElevenReader-style)

**Opportunity:** A single app for ADHD devs who:
1. Capture ideas/code thoughts by voice while coding
2. Dump project ideas without organizing
3. Feed ebooks/docs to be read aloud (eyes-free, hyperfocus-friendly)
4. Get AI to extract action items from brain dumps
